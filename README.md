# 中国航空博物馆分布图

一个交互式的中国地图应用，展示全国航空博物馆的分布位置。用户可以通过缩放地图发现各地的航空博物馆，点击标记查看详细信息。

## 🎯 功能特性

### 核心功能

- **省份聚合显示** - 在较小缩放级别（≤5）时，以数字徽章显示各省份的航空博物馆数量
- **具体位置标记** - 放大地图（≥6级）后，显示每个博物馆的具体位置，带有脉冲动画效果
- **交互式查询** - 点击任何标记可查看博物馆的详细信息，包括名称、位置、地址和简介
- **平滑缩放** - 支持鼠标滚轮和按钮进行地图缩放，实时切换显示模式
- **响应式设计** - 适配各种屏幕尺寸，提供最佳用户体验

### 设计特色

- **中国风美学** - 采用古铜色、朱红色等传统配色方案
- **书法字体** - 标题使用"马善政"书法字体，展现中文优雅气质
- **细节动画** - 标记脉冲动画、卡片淡入效果等微交互增强视觉体验
- **传统纹理** - 背景采用细微的水墨纹理，营造古韵氛围

## 🗺️ 数据结构

### 博物馆数据 (`museums.json`)

```json
{
  "museums": [
    {
      "id": 1,
      "name": "中国航空博物馆",
      "province": "北京市",
      "city": "朝阳区",
      "lat": 40.1234,
      "lng": 116.5678,
      "address": "北京市朝阳区赤港路...",
      "description": "中国唯一的国家级航空专业博物馆..."
    }
  ]
}
```

### 字段说明

| 字段 | 类型 | 说明 |
|------|------|------|
| id | number | 博物馆唯一标识 |
| name | string | 博物馆名称 |
| province | string | 所在省份 |
| city | string | 所在城市 |
| lat | number | 纬度坐标 |
| lng | number | 经度坐标 |
| address | string | 详细地址 |
| description | string | 博物馆简介 |

## 📖 使用指南

### 基本操作

1. **查看聚合数据** - 打开应用时默认显示省份级别的博物馆数量
2. **放大地图** - 使用 `+` 按钮或鼠标滚轮放大，当缩放级别≥6时显示具体位置
3. **点击标记** - 点击任何标记弹出详细信息窗口
4. **查看详情** - 在右侧面板查看选中博物馆的完整信息
5. **关闭面板** - 点击右上角的 `×` 关闭详情面板

### 缩放级别说明

| 缩放级别 | 显示内容 | 说明 |
|---------|---------|------|
| 3-5 | 省份聚合 | 显示每个省份的博物馆总数 |
| 6-10 | 具体位置 | 显示每个博物馆的精确位置 |

## 🎯 交互流程

```
应用启动
    ↓
显示中国地图（缩放级别4）
    ↓
[用户选择]
    ├─ 点击省份标记 → 显示该省份的博物馆列表
    ├─ 放大地图 → 切换到具体位置显示
    └─ 点击博物馆标记 → 右侧面板显示详细信息
```

## 🌐 地图配置

### 瓦片源

应用使用 OpenStreetMap 提供的免费地图瓦片：

```
https://{s}.tile.openstreetmap.org/{z}/{x}/{y}.png
```

### 地图边界

- **最小缩放级别** - 3（显示全国）
- **最大缩放级别** - 10（街道级别）
- **初始中心** - 中国地理中心（约35.86°N, 104.20°E）
- **初始缩放** - 4（全国概览）

## 🔄 更新与维护

### 添加新博物馆

1. 编辑 `client/public/museums.json`
2. 添加新的博物馆对象到 `museums` 数组
3. 确保包含所有必需字段
4. 重启开发服务器

### 更新坐标

如果需要调整现有博物馆的坐标：

1. 在 `museums.json` 中找到对应的博物馆
2. 更新 `lat` 和 `lng` 字段
3. 保存文件后自动刷新
