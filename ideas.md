# 中国航空博物馆地图 - 设计理念

## 设计方案选择

经过深思熟虑，我选择了**现代地理信息可视化**风格，结合**航空主题**的专业感与**交互探索**的趣味性。

---

## 选定方案：现代地理信息美学

### 设计运动
**现代地理信息可视化 + 航空工业美学**
- 灵感来源：Google Maps、高德地图等现代地理信息平台
- 融合航空工业的精密感与科技感
- 强调数据的清晰展示与交互的流畅性

### 核心原则
1. **清晰的信息层级** - 省份聚合数字清晰可见，缩放后显示具体位置
2. **流畅的交互体验** - 平滑的缩放、点击反馈、信息卡片动画
3. **专业的视觉语言** - 使用航空蓝、科技灰等颜色，传达专业感
4. **地理准确性** - 基于真实坐标的精确定位，尊重地理信息

### 色彩哲学
- **主色：航空蓝** (`#1e40af` / `#1e3a8a`) - 代表天空、飞行、信任
- **辅色：科技灰** (`#6b7280` / `#4b5563`) - 代表专业、稳定
- **强调色：橙色** (`#f97316`) - 代表活力、标记点
- **背景：浅色系** - 白色/浅灰，确保地图清晰可读
- **文字：深灰/黑色** - 高对比度，易于阅读

### 布局范式
- **全屏地图为主体** - 地图占据整个视口
- **顶部控制栏** - 包含标题、搜索、缩放控制
- **右侧信息面板** - 点击标记后显示博物馆详情
- **底部状态栏** - 显示当前缩放级别、鼠标坐标等信息

### 签名元素
1. **聚合数字徽章** - 圆形徽章显示每个省份的博物馆数量（如"2"）
2. **脉冲动画标记** - 具体位置的标记点有轻微脉冲效果
3. **信息卡片** - 点击后显示的博物馆详情卡，带阴影和过渡动画

### 交互哲学
- **渐进式信息披露** - 从省份聚合 → 缩放查看具体位置 → 点击查看详情
- **实时反馈** - 鼠标悬停显示提示，点击有明确反馈
- **平滑过渡** - 所有状态变化都有流畅的动画

### 动画指南
- **缩放动画** - 0.3-0.5秒的平滑缩放
- **标记脉冲** - 1.5秒循环，透明度 0.4 → 1 → 0.4
- **卡片出现** - 0.2秒淡入 + 轻微向上移动
- **悬停效果** - 标记放大 1.2x，阴影增强

### 排版系统
- **标题字体** - `Geist` 或 `Inter` 700 - 用于地图标题、省份名称
- **正文字体** - `Inter` 400/500 - 用于博物馆名称、描述
- **数字字体** - `Geist Mono` 或 `Inter` 600 - 用于聚合数字、坐标
- **层级结构**：
  - H1：24px 700 - 页面标题
  - H2：18px 600 - 省份名称
  - Body：14px 400 - 博物馆名称、描述
  - Caption：12px 500 - 坐标、辅助信息

## 实现细节

### 颜色变量（CSS）
```css
--color-sky-blue: #1e40af;      /* 主色 - 航空蓝 */
--color-dark-blue: #1e3a8a;     /* 深蓝 - 强调 */
--color-tech-gray: #6b7280;     /* 科技灰 */
--color-accent-orange: #f97316; /* 强调橙 */
--color-bg-light: #f9fafb;      /* 浅背景 */
--color-border-light: #e5e7eb;  /* 浅边框 */
```

### 组件设计
- **地图容器** - 全屏，响应式
- **省份聚合标记** - 圆形徽章，显示数字
- **具体位置标记** - 图钉样式，带脉冲动画
- **信息卡片** - 圆角卡片，带阴影，包含博物馆名称、位置、简介
- **控制栏** - 顶部固定，包含标题、搜索框、缩放按钮

---

## 设计决策理由

1. **为什么选择蓝色？** - 蓝色与天空、飞行紧密相关，同时是地图应用的标准色
2. **为什么是全屏地图？** - 地图是主要内容，应该占据最大空间
3. **为什么用聚合数字？** - 快速传达信息密度，符合用户的快速扫描习惯
4. **为什么有脉冲动画？** - 轻微的动画吸引注意力，但不会造成视觉干扰
5. **为什么右侧信息面板？** - 避免遮挡地图，保持地理信息的连续性

---

## 响应式设计
- **桌面** (≥1024px) - 全屏地图 + 右侧面板
- **平板** (768-1023px) - 全屏地图 + 底部抽屉式面板
- **手机** (<768px) - 全屏地图 + 全屏模态框显示详情
